## Amaze

An Open Source, light and smooth file manager that follows material design guidelines. It allows to work on multiple tabs at the same time, quickly access history, bookmarks and to search for any file.

![Star count badge for Amaze](https://badgen.net/github/stars/TeamAmaze/AmazeFileManager) ![Last commit badge for Amaze](https://img.shields.io/github/last-commit/TeamAmaze/AmazeFileManager)

## lrkFM

Free and open source file manager for Android with focus on archive extraction (.tar, .tar.gz, .zip, .7z and .rar (but not RARv5)) and creation (.zip).

![Star count badge for lrkFM](https://badgen.net/github/stars/lfuelling/lrkFM) ![Last commit badge for lrkFM](https://img.shields.io/github/last-commit/lfuelling/lrkFM)

**Links:** [Source repository](https://github.com/lfuelling/lrkFM) | [F-Droid](https://apt.izzysoft.de/fdroid/index/apk/io.lerk.lrkFM) | [Play Store](https://play.google.com/store/apps/details?id=io.lerk.lrkFM)
